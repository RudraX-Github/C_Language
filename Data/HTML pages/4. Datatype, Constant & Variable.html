1. Primary Data Types in C

C is a statically typed language. This means you must explicitly tell the compiler what kind of data a variable will hold before you use it. The Data Type determines two crucial things: How much memory (bytes) to allocate, and how to interpret that binary data.

int

Integer (Whole Numbers)

4 Bytes (32 bits)

Stores whole numbers without decimals. Can be positive or negative.

Format Specifier %d or %i
Value Range -2,147,483,648 to
2,147,483,647
int age = 25;
int temperature = -5;

float

Floating Point (Decimals)

4 Bytes (32 bits)

Stores numbers with fractional parts. Provides precision up to 6-7 decimal places.

Format Specifier %f
Precision 6 decimal places
float pi = 3.14159f;
float weight = 65.5f;

char

Character

1 Byte (8 bits)

Stores a single character, letter, number, or ASCII symbol. Must be enclosed in single quotes.

Format Specifier %c
Under The Hood Stored as integer (ASCII)
char grade = 'A';
char symbol = '#';

double

Double Precision Float

8 Bytes (64 bits)

Stores numbers with fractional parts, offering exactly twice the precision (memory) of a float.

Format Specifier %lf
Precision 15 decimal places
double exactPi = 3.141592653589793;

2. Memory, Variables & Constants

Now that you know the Data Types, you can declare Variables (named memory locations). The compiler uses the Data Type to know how many bytes to grab in RAM.

Rules for Naming C Variables

  • Must start with a letter (A-Z, a-z) or an underscore (_).
  • Can only contain letters, numbers (0-9), and underscores. No spaces!
  • Cannot be a reserved C keyword (e.g., int, return).
  • They are case-sensitive (age and Age are different).

Keywords are reserved words in C that cannot be used as variable names. Constants are variables declared with the const keyword; their values become locked and cannot be changed after initialization.

3. Format Specifiers & Printf()

Unlike Python or JS, C's printf() cannot automatically figure out what type of variable you are trying to print. You must use Format Specifiers as placeholders inside the string to tell C exactly how to format the raw memory data into text.

Specifier Cheat Sheet

  • %d Integer (decimal)
  • %f Float (decimal point)
  • %lf Double
  • %c Character
  • %.2f Float (2 decimal places)
printf_visualizer.c
int age = 25;
float height = 5.9;
char grade = 'A';
printf(" ", age, height, grade);

4. Scanf() & The Address-of Operator (&)

The scanf() function takes input from the keyboard. But to store that input, it needs to know exactly where in the computer's memory the variable lives. We use the Address-of Operator (&) to give scanf the physical memory address, not the value.

int userAge; // Currently holds garbage value
printf("Enter your age: ");
scanf("%d", &userAge);
