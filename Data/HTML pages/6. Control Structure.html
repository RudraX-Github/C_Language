3. Ternary Operator ( ? : )

The conditional operator is the only operator in C that requires three operands. It is used as a highly condensed, one-line substitute for simple if-else statements, often used in variable assignments.

Syntax Formula

Condition ? Value_If_True : Value_If_False ;

Interactive Test

// Find maximum value
