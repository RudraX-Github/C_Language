1. "Without Space" Patterns

A "Without Space" pattern is simply a left-aligned shape. Because the C console naturally prints from left to right, you only need two nested loops to create these. The outer loop manages the rows (i), and the inner loop manages the columns/characters (j).

left_triangle.c
int rows = 4;

for (int i = 1; i <= rows; i++) {
for (int j = 1; j <= i; j++) {
printf("*");
}
printf("\n"); // Move to next line
}

2. "With Space" Patterns

The Third Loop Secret To align a shape to the right (or center a pyramid), you must physically print "empty" space characters before printing the stars. This requires a third loop for(k) nested inside the outer loop, executing before the character loop.

The Math (Rows = 4)

Row i=1 Spaces = 3 Stars = 1
Row i=2 Spaces = 2 Stars = 2
Row i=3 Spaces = 1 Stars = 3
Row i=4 Spaces = 0 Stars = 4
Formula: Spaces = (rows - i)
// right_triangle.c
for (int i = 1; i <= rows; i++) {
// Print Spaces first
for (int k = 1; k <= (rows-i); k++) {
printf(" ");
}
// Print Stars second
for (int j = 1; j <= i; j++) {
printf("*");
}
printf("\n");
}

3. Custom Patterns (Numbers & Characters)

Instead of hardcoding a star "*", you can print the loop variables themselves. Printing the outer variable i gives you identical rows. Printing the inner variable j gives you sequences. You can also cast them to char to print alphabets!

Print Outer (i)

for(int i=1; i<=4; i++)
  for(int j=1; j<=i; j++)
    printf("%d", i);
1
22
333
4444

Print Inner (j)

for(int i=1; i<=4; i++)
  for(int j=1; j<=i; j++)
    printf("%d", j);
1
12
123
1234

Characters (ASCII)

for(int i=1; i<=4; i++)
  for(int j=1; j<=i; j++)
    printf("%c", 64+i);
