1. One-Dimensional (1D) Arrays

An array is a collection of elements of the same data type stored in contiguous (side-by-side) memory locations. In C, arrays are strictly 0-indexed, meaning the first element lives at index 0.

array_1d.c
// 1. Declaration & Initialization
int marks[5] = { };
// 2. Array Size Details
int sizeOfElement = sizeof(int); // 4 bytes
int totalMemory = sizeof(marks); // 20 bytes
// 3. Element Access (Read/Write)
printf("%d", marks[ ]);

The "Matrix" Illusion

2. Matrices & Row-Major Flattening

We logically visualize a 2D array as a table with Rows and Columns. But here is the hard truth: Your computer's RAM is NOT a grid. RAM is a flat, 1D line.

C stores 2D matrices sequentially by placing one row immediately after the other. This is called Row-Major Order. The compiler uses a math formula to instantly convert your 2D grid[row][col] coordinate into a 1D flat memory address.

matrix_2d.c
// 1. Declare 3x4 Matrix
int grid[3][4] = {
  {}, // R0
  {}, // R1
  {} // R2
};
// 2. Total Memory Space
Rows * Cols = 12 elements
Total = 12 * 4 bytes = 48 bytes
// 3. Access Coordinate
grid[][]
